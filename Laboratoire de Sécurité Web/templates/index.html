Projet de Fin d'Études en Sécurité des Applications Web

Cette application contient deux versions complètes du même portail étudiant : une version volontairement vulnérable pour les tests en laboratoire, et une version sécurisée pour démontrer la remédiation.

Mode vulnérable

Utilisez cette partie pour démontrer l'injection SQL, le XSS stocké, l'IDOR, le contrôle d'accès cassé, le stockage faible des mots de passe et l'upload de fichiers non sécurisé.

Ouvrir le mode vulnérable

Mode sécurisé

Utilisez cette partie pour montrer les requêtes paramétrées, le hachage des mots de passe, l'autorisation par objet, la validation d'upload, les journaux d'audit et les en-têtes de sécurité.

Ouvrir le mode sécurisé
